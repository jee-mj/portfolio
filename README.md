## 📁 Project Structure

The source code is structured to provide a clear and intuitive navigation experience:

```text
/
├─ src/
│  ├─ components/
│  │  ├─ bg/
│  │  └─ content.astro
│  ├─ content/
│  │  └─ [component]/
│  │     └─ [section].md
│  ├─ layouts/
│  │  └─ Layout.astro
│  └─ pages/
│     └─ index.astro
└─ public/
   └─ assets/
      └─ favicon.ico
```
